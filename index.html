Projects

Sky Tunes is an application that takes the weather of any city and a song given by the user to create a list of songs based off that given information.

In this project, my goal was to implement two APIs into my project. However, I ended up using three: Mapbox API, Weather API, and the Spotify API. One of the biggest challenges I faced was implementing the Spotify API because I had yet had experience dealing with servers. In order to jump over this hurdle, I watched YouTube videos to understand how it should be applied.

Created with: HTML, CSS, JavaScript

Resources used for project:

  • Weather API
  • Spotify API
  • Mapbox API
  • Background Image by Jerry Zhang
  • Google Fonts

Alphabet Soup! is a typing game where you are given a limited amount of time and a list of random words. The goal of the game is to see how many words you can type correctly within that given time.

In this project, my goal was create a fun, memorable experience using everything I've learned in IGME 235. My goal with this project was also to familiarize myself with PIXIjs. One challenge I encountered with this project was working with the keypress handler. Since most of my code was based off this event, I had to be aware of how I went about finding solutions. I had a difficult time structuring the code in a way that worked with this event handler.

Created with: HTML, CSS, JavaScript, PIXIjs, Canva

  • freesound.org

Entertainment Trivia is a trivia game where you can test your knowledge on all things entertainment! This site allows you to personalize your test so that you can choose the number of questions, the difficulty of questions, and the sub-category your questions will be based on!

In this project, my goal was to practice using an API while making a fun, memorable experience. My process was first by adding all the personalization controls to the html as well as empty containers that will be filled with test questions and test results. Next, I worked on the functionality of the site with JavaScript. I added event listeners for both buttons. The start button displays test questions based on the user’s personalization. The submit button displays the user’s results as well as the correct answers to the questions they got wrong. Lastly, I styled the site with CSS. I didn’t do anything too crazy with the styling, but I did create a background image for the title and results using Canva. I also made my site more in-tune with my theme by changing the typefaces.

Created with: HTML, CSS, JavaScript, Canva

Resources used for project:

  • Trivia API
  • Google Fonts

Tiger Mingle is an social media application that allows RIT students to meet new people on campus. Users will be randomly paired up with other users and will have the ability to message them. Once they reach a certain number of messages sent to one another, they will be able to play games like hangman, tic tac toe, and battleship.

This application was a group project in my IGME 201 class. I worked with 2 other student on the project for the duration of a semester. In this project, I took care of the chatting/messaging function, the random pairing generator, the hangman game, and the visuals.

Created with: C#, Windows Forms
