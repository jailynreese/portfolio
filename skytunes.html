Sky Tunes is an application that takes the weather of any city and a song given by the user to create a list of songs based off that given information. In this project, my goal was to implement two APIs into my project. However, I ended up using three: Mapbox API, Weather API, and the Spotify API. One of the biggest challenges I faced was implementing the Spotify API because I had yet had experience dealing with servers. In order to jump over this hurdle, I watched YouTube videos to understand how it should be applied.

Created with:

Vanilla JavaScript
HTML & CSS
